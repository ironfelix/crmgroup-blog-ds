Кто такой CRM-маркетолог

CRM-маркетолог — специалист, который выстраивает систему коммуникаций с клиентами через email, SMS, мессенджеры и push. Его задача — удерживать клиентов и повышать их ценность для бизнеса.

Хороший CRM-маркетолог думает не письмами, а жизненным циклом клиента. Его главный вопрос: «что происходит с человеком между касаниями?»

Что он делает

Типичный список задач CRM-маркетолога в агентстве или инхаус:

  • Разрабатывает стратегию CRM-коммуникаций
  • Настраивает триггерные сценарии и автоматизацию
  • Пишет ТЗ на письма, контролирует дизайн и вёрстку
  • Анализирует результаты и делает A/B-тесты

Структура:
.navigation-page — оглавление
.wp-block-group — универсальный контейнер
.wp-block-group__inner-container — внутренний wrapper
.wp-block-columns / .wp-block-column — сетка колонок
.wp-block-fsb-flexible-spacer — адаптивный отступ

Блоки:
.block-comment — цитата/комментарий эксперта
.block-comment__text — текст цитаты
.block-comment__autor — строка автора
.autor-photo / .autor-name / .autor-post — данные автора
.block-result — CTA-блок с результатом
.block-result__heading — заголовок CTA (.h2-smalls)
.block-form — форма подписки
.block-picture / .picture — обёртка изображения

Типографика:
.wp-block-heading — заголовки H2/H3
.wp-block-list — списки ul/ol
.wp-block-table / .table — таблицы
.has-inline-color.has-vivid-red-color — красный текст
.h2-smalls — уменьшенный H2 (для CTA)

Кнопки:
.wp-block-buttons — контейнер кнопок
.wp-block-button — обёртка кнопки
.wp-block-button__link — сама кнопка (тег <a>)
.social-networks / .social-networks__button — соцсети
.social-networks__button.vk / .telegram — конкретная сеть

Модификаторы:
.has-background — блок с фоном
.alignfull — блок на всю ширину
.is-layout-flex / .is-layout-flow / .is-layout-constrained
.is-style-regular / .is-style-default — стили таблиц
data-anchor="anchor_N" — якорь для TOC

  document.body.insertAdjacentHTML('afterbegin',
    '<div class="reading-progress"><div class="reading-progress__bar" id="rpBar"></div></div>');
  window.addEventListener('scroll', () => {
    const s = document.documentElement;
    const pct = s.scrollTop / (s.scrollHeight - s.clientHeight) * 100;
    document.getElementById('rpBar').style.width = pct + '%';
  }, { passive: true });
});
