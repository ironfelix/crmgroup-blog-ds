Что такое LTV и как его считать?
LTV (Customer Lifetime Value) — общая прибыль от одного клиента за всё время взаимодействия. Формула: LTV = ARPU × средняя продолжительность взаимодействия. Например, клиент платит 500 рублей в месяц в течение года: LTV = 500 × 12 = 6 000 рублей.
Что такое Retention Rate и какой показатель считается нормальным?
Retention Rate (RR) — процент клиентов, которые продолжают пользоваться продуктом. Нормальный уровень зависит от отрасли и сравнивается внутри компании по периодам, а не с конкурентами.

Пустые картинки — CSS :has() новый

CSS :has() автоматически скрывает блоки с пустыми или отсутствующими изображениями на хабе и главной странице. Без JS.

/* Скрыть блок автора если img пуст или отсутствует */ .new_article__author:has(img[src=""]), .new_article__author:has(img:not([src])), .new_article__author:has(.new_article__author-photo:empty) { display: none !important; } /* Слайд без картинки: текстовый блок на всю ширину */ .new_article:has(.new_article__author-photo:empty) .new_article__text, .new_article:has(img[src=""]) .new_article__text { width: 100% !important; } /* Скрыть пустой card-article__visual (главная) */ .card-article__visual:has(img[src=""]), .card-article__visual:has(img:not([src])) { display: none !important; } /* Мобилка: убираем фиксированный height */ @media (max-width: 990px) { .new_article:has(.new_article__author-photo:empty) .new_article__text { height: auto !important; min-height: 280px !important; } }
CSS :has() поддерживается в Chrome 105+, Safari 15.4+, Firefox 121+. Для старых браузеров — graceful degradation (блок остаётся видимым).

5 ключевых метрик для измерения эффективности CRM-стратегии

Пять показателей, которые помогут увидеть общую картину взаимодействия с аудиторией и выявить возможности для роста дохода.

Содержание
LTV — пожизненная ценность Retention Rate Churn Rate

LTV — пожизненная ценность клиента

Как считать

LTV (Customer Lifetime Value) показывает общую прибыль от одного клиента за всё время взаимодействия. ARPU (Average Revenue Per User) — средний доход от одного клиента за период.

Главная трудность с подсчётом LTV — определиться, на каком интервале это делать. По определению, надо считать весь срок жизни клиента.
