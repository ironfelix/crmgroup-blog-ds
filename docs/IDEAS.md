# Идеи для CRM-group

---

## Страницы по отраслям

**Идея:** Создать отдельные landing-страницы под каждую отрасль — как у конкурентов (WIM, Out of Cloud).

**Зачем:** Рынок не специализирован — никто из топ-агентств не занял отраслевые ниши. Отраслевые страницы = SEO + конверсия (клиент видит «они работали с такими как я»).

**Пример конкурента:** WIM.agency — раздел «Отрасли»: Недвижимость, Образование, B2B, Медицина, Финансы, Телеком.

**Предлагаемые отрасли для CRM-group:**
- E-commerce / Ритейл (уже есть кейсы — стартовая точка)
- EdTech / Образование
- HoReCa / Рестораны / Доставка
- Фармацевтика / Медицина
- Финансы / Банки / Страхование
- Недвижимость

**Что должно быть на каждой странице:**
- Специфика CRM в этой отрасли (боли, метрики)
- Кейс с цифрами из этой отрасли
- Типовые сценарии автоматизации
- CTA под отраслевую специфику

**Источник инсайта:** конкурентный анализ февраль 2026 → `competitors-analysis.md`

## Лид-магниты

Стратегия: вместо «оставьте заявку» предлагаем бесплатную ценность → собираем базу → nurturing по email → заявка. Использовать и в органике, и в контексте.

### Приоритет 1 — делать первыми

**1. Калькулятор ROI email-канала**
- Интерактив на сайте: вводишь базу, частоту, средний чек → получаешь потенциал выручки
- Первый такой инструмент в РФ среди email/CRM-агентств
- Для органики: посадочная страница под запрос «сколько стоит email маркетинг»
- Для контекста: «Посчитай сколько теряешь без CRM-маркетинга»

**2. Автоматический аудит email-маркетинга**
- Квиз 15-20 вопросов → автоматический PDF-отчёт на email (без звонка продажника)
- Не «оставьте заявку», а реальная ценность
- Для контекста: «Бесплатный аудит за 5 минут»

### Приоритет 2 — после запуска

**3. Пак шаблонов триггерных писем**
- Welcome-серия, брошенная корзина, win-back — скачиваемый ZIP
- Высокий спрос органики: «шаблоны email рассылок», «шаблон email письма»
- Скачивание за email

**4. Индекс email-маркетинга в России 2026**
- Закрытый PDF-отчёт, доступ за email
- Публиковать на emailsoldiers.ru (независимый вид)
- Для контекста: аудитория CMO, директора по маркетингу

**5. Гайд «CRM-маркетинг для ритейла»**
- Стратегический гайд для CMO среднего ритейла (не для специалистов)
- Длинный nurturing-цикл
- Закрытый PDF, 20-30 страниц

### Воронка
```
Контекст / Органика
        ↓
    Лид-магнит (бесплатно)
        ↓
    Email-база (nurturing 4-6 писем)
        ↓
    Заявка / Демо
```

Дополнительно: лид-магниты можно использовать как кейс «мы сами так делаем» — показывает экспертизу на практике.
